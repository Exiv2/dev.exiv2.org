RE: Newbie - command line to create XMP sidecar from .CR2 file - Added by Alan Pater over 6 years ago

Works for me with just:

exiv2 -eX img_000001.cr2

I think it is required that the cr2 file actually has existing metadata. That can be checked with:
exiv2 -pa img_000001.cr2
