An Action or Command for Keeping Specified Tags

Added by Dan S over 7 years ago

I believe it would be useful to have some way to specify a list of tags to keep. Then all tags that don't match will be dropped.

For example, an image file may have a bunch of meta data (Exif, Xmp, and Iptc). A user may just want to keep a few of the Exif and Iptc tags.

The functionality I am imagining would work something like:
exiv2 -M"keep Exif.Image.Make Exif.Image.Model Iptc.Application2.DateCreated" image.jpg

In that case, only the tags with matching keys would be retained. All Xmp tags would be deleted and any Exif or Iptc tags that don't match those keys would also be deleted.

The list of tags could be arbitrarily long.

Please let me know if there is already some way to achieve this, and if not, should I go ahead and make a feature request by starting a new issue? I suppose that one way to achieve this is to write a script that prints all tags in the file with "exiv2 -p a pr", then loop through all the keys, and if the key is not on some predefined list, then issue a delete command with exiv2.

Thanks.

RE: An Action or Command for Keeping Specified Tags - Added by Robin Mills over 7 years ago

Dan

I think you've answered your own question. Write a script. It's probably a one-liner in bash. Something like:

f=foo.jpg ; exiv2 -pt "$f" | cut -d' ' -f 1 | grep -v Exif.Image.Make | grep -v Exif.Image.Model

I might be a little off with the syntax. But you get the idea. exiv2 -pt foo.jpg tells you all the tags and values, cut -d' ' -f 1 filters to only pass the tags. grep -v filters out the ones you want to keep. So now you have your kill list. You can use the dentist's method (pull them out, one at a time), to delete the tags.

f=foo.jpg ; for tag in $(exiv2 -pt foo.jpg | cut -d' ' -f 1 | grep -v Exif.Image.Make | grep -v Exif.Image.Model ); do exiv2 -M"del $tag" "$f" ; done

I'm sure you can use sed to format the tags to be deleted into a command of the form:
exiv2 -M"del Exif.GPSInfo.GPSLongitudeRef GPSVersionID" R.jpg

I'm sure you can put the tags into a comma separated list and in the end you have a clever little script:
exiv2keep.sh "Exif.Image.Make,Exif.Image.Model" R.jpg

If you're using CMD.exe, install Cygwin to get bash. If you're stuck with CMD.exe, the web site "Dos Tricks and Tips" is a good resource. Perl and Python are also very good for dealing with this kind of thing and will usually work without change on both bash and CMD.exe.

Steve Wright is a regular visitor to the forum and really good with scripting shenanigans. Steve might have something to say about this.
