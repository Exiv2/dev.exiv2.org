Roadmap

Added by Andreas Huggel over 12 years ago

The target date for the roadmap for 0.19 as published is no longer realistic, I'm afraid. I haven't had enough time to work on Exiv2 recently, and that won't change anytime soon. As usual, that affects mainly larger features, not smaller fixes and changes, thus I have decided to re-arrange the roadmap accordingly.

The 0.19 roadmap as it exists will become the roadmap for a future version 1.0, but without a fixed release date. In the meantime, there will be smaller bugfix and minor feature releases. This will ensure that new releases can be made as usual, instead of having to wait for a major update for a long time.

I think users will benefit from such incremental releases. The backdraw is that the API/ABI will continue to evolve too, which can be painful for distros and application developers. I'll try to limit interface changes to only what is really necessary and to as few releases as possible. Essentially it means that this project continues to live a beta-life for now, where such changes do happen.

Andreas
