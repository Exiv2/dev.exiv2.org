Nikkor name scheme

Added by Torsten Bronger about 8 years ago

Hi! I work on the database of the LensFun project. You can see in our list of supported lenses that most Nikkor lenses don't have the string "Nikon" in their name. However, at least for the NEF files that I got (I don't own a Nikon), exiv2 returns a name following the scheme

Nikon AF(-S)? (Zoom-)?Nikkor ...

Most programs using LensFun rely on exiv2. I wonder whether it is sensible to go through all Nikkors and adjust the name according to this scheme. What do you think?


Replies (3)

RE: Nikkor name scheme - Added by Niels Kristian Bech Jensen about 8 years ago

In Exiv2 (and ExifTool) we try to follow the names given by Nikon (and other manufacturers).

Lensfun auto search seems to work fine with the current name scheme. The question is if the change you suggest is necessary? It should not hurt anything though.

Best regards,
Niels Kristian Bech Jensen

RE: Nikkor name scheme - Added by Torsten Bronger about 8 years ago

LensFun allows too many words in the LensFun lens model name, but it rejects too many words in the EXIF model name. For example, it a lens is called "Nikkor 17-55mm ..." in LensFun, but exiv2 says "Nikon Nikkor-Zoom 17-55mm ...", LensFun won't return a match.

So I guess it makes sense to make LensFun's Nikkor names more verbose ...

RE: Nikkor name scheme - Added by Torsten Bronger almost 8 years ago

Apparently, Nikon has changed it naming scheme some time ago. Quite recent lenses like [[http://www.nikonusa.com/en/Nikon-Products/Product/Camera-Lenses/2163/AF-S-NIKKOR-14-24mm-f%252F2.8G-ED.html]] do not contain the name "zoom" anymore in their name. However, exiv2 calls this lens "Nikon AF-S Zoom-Nikkor 14-24mm f/2.8G ED".

So, what does the exiv2 team think is the best approach for LensFun to go? Sticking to one scheme for all Nikkors or use the name from Nikon's webpage?
