New Canon CR2 write-support code needs some testing

Added by Andreas Huggel over 9 years ago

I've added some experimental code to write to CR2 images, finally (#635). Ufraw (i.e., dcraw) can still open a modified image and the OSX image viewer still shows the image (not sure if that's only a preview though), other than that I've not tested it much, in particular I do not have any Windows or Canon software to check with.

You can help test by checking-out the latest trunk version of Exiv2, using it to update some metadata in CR2 images and checking if your favorite image software still recognizes the modified images afterwards. Please report your success or any issues back here.

Do make sure you have a backup of your original CR2 images before modifying them with Exiv2!

Andreas
