RE: compile exiv2 from trunk - Added by Andreas Huggel over 12 years ago

That sounds like you're missing the gettext tools. Alternatively, you can run configure with the --disable-nls option. See the section 'Dependencies' in the README. In either case you'll need to run configure again or start from scratch (make maintainer-clean; make config).

-ahu.
