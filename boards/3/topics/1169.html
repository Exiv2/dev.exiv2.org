Custom Exif tags using exiv2

Added by Senan V over 9 years ago

Hi,

I am working in an image processing project and I need to write some color correction details as metadata in jpeg file for future reference.Is there any way to create a custom exif tag using exiv2 ?

Replies (2)

RE: Custom Exif tags using exiv2 - Added by Robin Mills over 9 years ago

I've looked at the exif specification: http://exif.org/Exif2-2.PDF and I don't see anything that seems designed for your purpose. The UserComment is a string and could be used for this purpose. Of course, this field may be changed in the workflow and your data lost.

Every exif "object" contains a unique identifier. For example:
Exif.Photo.ImageUniqueID Ascii 33 f861f2420a5fe2828e6671d168e3a0b5

Depending on your work flow, you could store the color correction details in an external database and use ImageUniqueID as a key in your database.

The XMP specification: http://partners.adobe.com/public/developer/en/xmp/sdk/XMPspecification.pdf may provide suitable hooks. As XMP is "Extensible Metadata Platform", the name implies that you may add your own metadata.
