RE: Command Line: Writing JP2 XMP Metadata Problem - Added by Andreas Huggel almost 13 years ago

Hi Cami,

I can write to the attached file using the Windows executable from the Exiv2 website:

OATEST-02> ./exiv2 -M'set Xmp.dc.Source 123' 1.jp2 
OATEST-02> ./exiv2 -pa 1.jp2 
Xmp.dc.Source                                XmpText     3  123
  • Which version of Exiv2 are you using? You need 0.18 for JP2 write support. Check with exiv2 -V
  • With 0.18, does the simple example above work for you?

-ahu.
