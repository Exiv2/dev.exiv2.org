Merged gsoc2012 branch into trunk

Added by Andreas Huggel over 9 years ago

The coding period for the Google Summer of Code 2012 programme ends today. Abhinav Badola has been working on this programme for the past three months and has done a good job implementing video metadata support (feature #813) for the most popular video formats. I've now tagged the final status of the gsoc2012 branch (r2851) and merged it into the trunk (r2852). Further video metadata work will take place directly in the trunk, Abhinav has indicated interest to continue, and there is certainly more work to polish what we have now as well as to extend the video metadata support further.

Expect a bit a rough ride for a moment, as the new code gets more exposure. Development has been done on Linux only as far as I know. I can compile and successfully run the test suite on my Debian system, but compilation on other platforms, esp. Windows, may now be broken.

Please report related issues here or as proper issues and bear with us.

Andreas
