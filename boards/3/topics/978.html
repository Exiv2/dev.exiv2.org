Release 0.22 is out!

Added by Andreas Huggel about 10 years ago

This release introduces support for Postscript (EPS) images. XMP metadata can be read and written from/to EPS images and previews are accessible. Further it includes a new build environment for MSVC 64 bit builds and a number of smaller features and bugfixes.

Release 0.22 is not binary compatible with 0.21.1, however it's almost API compatible and re-compilation should be all that's required, I don't expect a need for any client code to be changed.

Thanks to all contributors, especially Volker and Michael who spent a lot of time implementing and testing support for all sorts of EPS flavors and to Robin for the new MSVC 64 bit build environment and his continued contributions for everything related to MSVC.

Andreas
