RE: Sigma 18-250mm f/3.5-6.3 DC Macro OS HSM - Added by Niels Kristian Bech Jensen almost 9 years ago

A comment in ExifTool told me that the second number (LensFStops) varies with the focal length setting of this lens (and maybe other Sigma zoom lenses). I have made a patch for the current exiv2 svn that ignores that number for Sigma lenses. Please try it and tell us if it solves the problem.

Regards,
Niels Kristian

RE: Sigma 18-250mm f/3.5-6.3 DC Macro OS HSM - Added by Niels Kristian Bech Jensen almost 9 years ago

Glad to help.

The second number in the extended lens ID is an encoding of the number of F-stops (aperture values) that can be set for each focal length (Lens F Stops in the ExifTool output).

Regards,
Niels Kristian
