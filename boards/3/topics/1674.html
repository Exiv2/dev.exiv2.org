Lens type for Canon PowerShot S90 reported as (65535)

Added by Mark R almost 8 years ago

I am trying to get raw images from my Canon PowerShot S90 properly developed with darktable. I have created a lensfun calibration file to correct distortion and tca (following the tutorial at http://wilson.bronger.org/lens_calibration_tutorial/). The file is stored at $HOME/.local/share/lensfun/canon-powershot-s90.xml with the following content:

<lensdatabase>
    <lens>
        <maker>Canon</maker>
        <model>Standard</model>
        <mount>canonS90</mount>
        <cropfactor>4.6</cropfactor>
        <calibration>
            <distortion model="ptlens" focal="6" a="0.01801" b="-0.09667" c="0.02041" />
            <distortion model="ptlens" focal="7.5" a="0.00793" b="-0.0524" c="-0.00156" />
            <distortion model="ptlens" focal="10.7" a="-0.01274" b="0.02428" c="-0.04366" />
            <distortion model="ptlens" focal="18.2" a="0.02233" b="-0.0766" c="0.07502" />
            <distortion model="ptlens" focal="22.5" a="-0.00256" b="0.01016" c="-0.01643" />
            <tca model="poly3" focal="6" br="0.0004594" vr="0.9997919" bb="-0.0004823" vb="1.0009686" />
            <tca model="poly3" focal="7.5" br="0.0002156" vr="0.9998923" bb="-0.0003540" vb="1.0009066" />
            <tca model="poly3" focal="10.7" br="0.0000737" vr="1.0000337" bb="-0.0001950" vb="1.0005510" />
            <tca model="poly3" focal="18.2" br="0.0000189" vr="1.0001247" bb="-0.0000461" vb="1.0001230" />
            <tca model="poly3" focal="22.5" br="-0.0000520" vr="1.0003164" bb="-0.0000220" vb="1.0000067" />
        </calibration>
    </lens>
</lensdatabase>

However, the lens correction module in darktable-1.2.3 does not automatically pick this entry. I noticed that the "image information" view in darktable shows "(65535)" as value for "lens".

So, as a workaround, I tried changing

  <model>Standard</model>

to

  <model>(65535)</model>

and now it works as expected. exiv-0.23 shows the following lens infos:

$ exiv2 -pt 20131110-151749.jpg | grep -i lens
Exif.CanonCs.LensType                        Short       1  (65535)
Exif.CanonCs.Lens                            Short       3  6.0 - 22.5 mm

It seems to me that if exiv2 would report Exif.CanonCs.LensType as "Standard" everything would be working fine. But I'm just guessing. Maybe someone with more insight into the whole exiv2 / lensfun /darktable chain can help me on this?

Thanks,
Mark
