RE: crash with 0.19 - Added by Andreas Huggel over 11 years ago

There was no API change in this area, this shouldn't crash.
I can think of these two possible causes if it does:

  • a specific image causes exiv2 to fall over
  • gthumb was compiled and linked with a different version of exiv2, e.g., SVN trunk, which is not binary compatible with 0.19. Re-compiling would definitely be required.
