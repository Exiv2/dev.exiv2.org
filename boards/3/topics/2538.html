RE: How does wstring support work for Exiv2::ImageFactory::open()? - Added by Robin Mills over 5 years ago

Mikayel

You have to build Exiv2 to support wstring paths. You set the flag EXV_UNICODE_PATH in exc_msvc.h.

I'll be honest, I've never built this because all our test applications require modification to work with wstring paths. Qt Applications normally use 16bit strings for paths and I believe it works. Here's a discussion I had in November with somebody about this. http://dev.exiv2.org/boards/3/topics/2311
