RE: Script offering -- read and write .exv files based on a list - Added by Steve Wright about 10 years ago

I fixed the script in the OP. But I also modified file2fields.sh to work with lists with more than one file.
The way it's written, it allows for one empty space between the last field to be added to one jpg and the file name of the next.
There's also a conditional loop meant to check whether the file read in by name exists in the folder, so you don't have to put up with several exiv2 errors that say basically the same thing.

Hope someone finds it useful. Modification for the purposes of reading in and tagging your preferred IPTC tags is certainly allowed. (That goes for both these scripts.)
