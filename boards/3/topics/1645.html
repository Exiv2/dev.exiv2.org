RE: lens Sigma 150 mm 1:2.8 APO MACRO DG HSM not reckognized - Added by Steve Fosdick about 8 years ago

The lens that is listed in your RTF document is one which is associated with lens ID 183. In the latest subversion trunk of exiv2 this is also associated with the Sigma lens you describe. Here are the lines from the Canon Maker Notes decoding file (canonmn.cpp):

{ 183, "Canon EF 100-400mm f/4.5-5.6L IS"                           }, // 0
        { 183, "Sigma 150mm f/2.8 EX DG OS HSM APO Macro"                   }, // 1
        { 183, "Sigma 105mm f/2.8 EX DG OS HSM Macro"                       }, // 2

I don't know exactly which version of exiv2 you are using. It may be worth posting that here is you can find out or you could try grabbing the latest version from subversion and see if that solves the problem.
