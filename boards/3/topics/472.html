What is the proper way to enumerate schemas present in image file?

After reading the metadata, the XMP properties can be accessed through Image::xmpData()
There is no list with just the schemas (namespaces), you'll have to go through the XMP properties and build one yourself (e.g., by populating a map with Xmpdatum::groupName() )

See:
http://www.exiv2.org/doc/index.html
http://www.exiv2.org/doc/classExiv2_1_1Xmpdatum.html

How to enumerate schema datum in image file?

See http://www.exiv2.org/doc/xmpsample_8cpp-example.html the for-loop with the comment "Output XMP properties" towards the end.

How to remove the schema (with its datum) from the file?

Remove all XMP properties of that schema from the XmpData container and write the metadata back to the image.
