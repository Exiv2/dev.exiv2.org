Decoding makernote lens ID

Added by Anonymous Poster almost 11 years ago

Hello,

hope you can help me a bit. I'm trying to read the lens out of the makernote tag.

To read the lensID I use the following code which is working fine:

Exiv2::Image::AutoPtr Exiv2image = Exiv2::ImageFactory::open(gimp_image_get_filename(imageID));
 Exiv2image.get();
 Exiv2image->readMetadata();
 Exiv2::ExifData &exifData = Exiv2image->exifData();
LensID = exifData["Exif.Pentax.LensType"].toLong();

But how can I decode the lensID which is just a number to the string that identifies the lens. For example "SMC Pentax 12-24/2.8..."

Cheers,
Sebastian

RE: Decoding makernote lens ID - Added by Andreas Huggel almost 11 years ago

Sebastian,

First, I suggest you use the lensName() "easy-access" function instead of this particular tag. That will make your code work with images from other cameras as well. Then in order to get the lens name rather than some number, you need to apply the "pretty-print" function to the metadatum. That's what the Exifdatum::print() and Exifdatum::write() functions do. See this old thread for additional info: http://uk.groups.yahoo.com/group/exiv2/message/1408

Andreas
