RE: MakerNote - Added by Robin Mills over 6 years ago

Mikayel

Indeed we report the data in the maker note "in the raw", however I believe we decode the manufacturer proprietary data as best we can. It's described here:
http://www.exiv2.org/makernote.html There's an index of manufacturer specific tags here: http://www.exiv2.org/metadata.html

As you know I'm the build/test engineer and didn't write the code, so I can't explain exactly how it works. I recommend that you step the code in the Visual Studio debugger to see what is done. Please remember that "open source" is source and we encourage users to read the code to understand how it works. Every member of the team is an unpaid volunteer.

Robin
