Exiv2 is the C++ library and command line utility to manage image metadata. It provides fast and easy read and write access to the Exif, IPTC and XMP metadata of images in many formats.

From here you can access all project resources and interact with project members and Exiv2 users. By following a few simple guidelines you stand a good chance of getting an answer quickly. Such feedback is a valuable contribution to the project and will help to improve the quality of the software and provide directions for future development.

The main points to consider before contributing are:





    10
    1
    Andreas Huggel
    
# Search this site, the "old mailing list":http://uk.groups.yahoo.com/group/exiv2 and use your favourite search engine to check if your issue has been raised before or your question has been answered already.





    11
    14
    Andreas Huggel
    
# Decide whether to post to the forums, raise a new issue or add your comments to an existing bug, feature or forum post. If you have found a bug and you're reasonably sure that it really is a bug (e.g., if the exiv2 command line tool crashes), it's best to open an issue for it, with instructions how to reproduce the problem. If the issue is with a specific image, we'll most likely need the image. 





    12
    
    
    
# In any case, posting here is better than writing to any project member privately as this allows others to discuss the issue and supply answers which the person may not be able to give. Also, other users can later find these answers in the forum without having to ask again.





    13
    
    
    
# If you'd like to get involved in the development or contribute code, please post your ideas in the forum before sending a patch, unless it addresses a specific open issue.
