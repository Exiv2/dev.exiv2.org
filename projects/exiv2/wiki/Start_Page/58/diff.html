h1. Guidelines for contributing code Welcome to the Exiv2 community!

*Principles:*
# Make it as simple as possible

# Every time community website of the project. The Exiv2 homepage is http://www.exiv2.org/

From here you find can access all project resources and interact with project members and Exiv2 users.
