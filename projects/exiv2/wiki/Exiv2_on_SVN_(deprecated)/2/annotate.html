For until March 2017, the Exiv2 source code lived on svn://dev.exiv2.org/svn/trunk  When v0.26 is released, this resource will be _*frozen and never updated*_.  The revision history in SVN has been migrated to GitHUB.  Eventually the SVN repository will be removed when it's clear that it serves no further purpose.
