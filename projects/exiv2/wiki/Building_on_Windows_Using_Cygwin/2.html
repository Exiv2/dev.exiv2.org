Building on Windows Using Cygwin¶

  1. Install cygwin from https://cygwin.com/setup-x86_64.exe (64-bit) or https://cygwin.com/setup-x86.exe (32-bit)
  2. Choose the following packages: autoconf, clang, cmake, gcc, getext-devel, libexpat-devel, make, pkg-config, subversion, zlib-devel









  3. Install the selected packages
