h1. Accessing preview images
2

Since release 0.18, Exiv2 contains functionality to easily access preview images embedded in the metadata (Exif, IPTC or XMP).





    4
    1
    Andreas Huggel
    





    5
    13
    Andreas Huggel
    
RAW images may contain several (I've seen up to four) preview images in addition to the main image in different sizes and usually in JPEG or TIFF format. They range from tiny thumbnails to full-sized JPEG images.

    Andreas Huggel

Applications can use the preview list to select which preview image is suitable and work with a preview of the actual picture without having to extract and potentially decode the large main image.

For examples of how to access the preview images with the Exiv2 command line tool, see Franz Buchinger's blog entry "Ultra-fast RAW to JPEG conversion with exiv2":http://www.picurl.org/blog/2009/07/18/ultra-fast-raw-to-jpeg-conversion-with-exiv2/.

For programmers, the main API of the new functionality is "@Exiv2::PreviewManager@":http://www.exiv2.org/doc/classExiv2_1_1PreviewManager.html. The interface is designed to make it transparent for applications where and how exactly the preview images are stored in the metadata. It has little overhead in order to allow fast access to even the tiniest thumbnail.
