Exiv2 on SVN (deprecated)¶

Until March 2017, the Exiv2 source code lived on svn://dev.exiv2.org/svn/trunk When v0.26 is released, this resource will be frozen and never updated. The revision history in SVN has been migrated to GitHUB. Eventually the SVN repository will be removed when it's clear that it serves no further purpose.

There are other directories stored in the repos:

testdata svn://dev.exiv2.org/svn/testdata Additional test assets for video, eps and other features
team svn://dev.exiv2.org/svn/team team assets such as the Exiv2 logo graphics
branches svn://dev.exiv2.org/svn/branches a collection of development branches (mostly historic)
trunk svn://dev.exiv2.org/svn/trunk Primary Source Resource
