h1. Building on MacOSX Using Autotools
2 





    3
    
    
    





    4
    
    
    
* This has been tested on Mac OS X 10.9.4 Build: 13E28





    5
    
    
    
* Last revision on which this method was tested: r3348





    6
    
    
    
* This wiki page assumes that installation prefix is _/usr/local_ and your build-dir is _~/gnu/exiv2_





    7
    
    
    





    8
    
    
    
# Install gcc





    9
    
    
    
<pre>





    10
    
    
    
sudo port install gcc49





    11
    
    
    
</pre>





    12
    
    
    
# Fetch the source code:





    13
    
    
    
<pre>





    14
    
    
    
mkdir -p ~/gnu/exiv2





    15
    
    
    
cd ~/gnu/exiv2





    16
    
    
    
svn checkout svn://dev.exiv2.org/svn/trunk





    17
    
    
    
cd trunk





    18
    
    
    
</pre>





    19
    
    
    
# Build exiv2





    20
    
    
    
<pre>





    21
    
    
    
make config





    22
    
    
    
./configure CC=gcc-mp-4.9 CXX=g++-mp-4.9  # to use clang: ./configure CC=clang CXX=clang++





    23
    
    
    
make                                      # make CXXFLAGS=-ggdb for debug





    24
    
    
    
sudo make install





    25
    
    
    
</pre>





    26
    
    
    
# Building samples





    27
    
    
    
<pre>





    28
    
    
    
export PKG_CONFIG_PATH=/usr/local/lib/pkgconfig





    29
    
    
    
make samples





    30
    
    
    
</pre>





    31
    
    
    
# Build tests.





    32
    
    
    
** To execute tests:





    33
    
    
    
<pre>





    34
    
    
    
cd ~/gnu/exiv2/trunk





    35
    
    
    
make tests





    36
    
    
    
</pre>





    37
    
    
    
** Running other "test suites":http://dev.exiv2.org/projects/exiv2/wiki/How_do_I_run_the_test_suite_for_Exiv2
