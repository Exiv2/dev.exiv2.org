# pkg-config is not part of standard MinGW installation. Assuming your @MinGW@ installation is in @C:\MinGW@, download "pkg-config":http://ftp.gnome.org/pub/gnome/binaries/win32/dependencies/pkg-config_0.23-3_win32.zip and extract @pkg-config.exe@ executable to @C:\MinGW\bin@. Then download "glib":http://ftp.gnome.org/pub/gnome/binaries/win32/glib/2.28/glib_2.28.1-1_win32.zip and extract liglib-2.0-0.dll to @C:\MinGW\bin@. Open command prompt and type:
