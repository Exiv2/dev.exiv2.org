Description

LensType tag is not reported for pictures taken with new Pentax K-x camera.
Example:

$ exiv2 -p a IMGP0147.JPG | grep LensType
Exif.Pentax.LensType Byte 4 7 231 0 0

Lens type is known, but 4 bytes are reported where you expect 2 bytes only.
