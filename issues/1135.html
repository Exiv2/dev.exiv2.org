Updated by Andreas Huggel about 6 years ago

Exiv2 can update any metadata tag that it reads (with some deliberate exceptions), including Makernote tags. All tags are updated the same way, so once you understand how to update a regular (Exif) tag, you can do the same thing with any makernote tag. In the example (see the output that Robin posted), the Exif.PentaxDng.* tags are the makernote tags. Try something like exiv2 -M'set Exif.PentaxDng.SerialNumber ABCDEF' to set a tag and see the Command line utility section on the Getting Started page on the website to learn more.
PS: Maybe a DNG image is not the best example for an image with a makernote, I suggest you use a JPEG image to experiment.
PPS: A question like this is better asked in the forum. If you file it as a bug, like you did, we expect you found an issue with the software; something that is not working as designed, as opposed to something you have not figured out yet.
