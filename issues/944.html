Doubled "AF" for some Tokina lens model names on Nikon cameras

Description

In nikonmn.cpp, there are things in parentheses in Tokina lens model names, for example,

AF 193 (AF 19-35mm f/3.5-4.5)

I don't really know how this is converted into a lens model in exiv2's output, but a doubled "AF" would make autodetection using lensfun difficult (loose matching would be necessary). So if possible, please remove the first "AF".
