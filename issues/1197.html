Description

The Sigma 150-500mm f/5-6.3 APO DG OS HSM lens is recognised on Canon both solo and with a 2x teleconverter but not with the 1.4x TC. In the +1.4x TC configuration it uses lens ID 172 and this patch adds the necessary table entries.

I do not have either of the Canon EF 400mm f/5.6L or the Sigma 150-600mm f/5-6.3 DG OS HSM | S so I cannot test if these are still correctly recognised after this patch, only that the Sigma 150-500mm f/5-6.3 APO DG OS HSM + 1.4x is.
