Feature #1190

Support for CRS and CRSS XMP namespace and properties

Description

Adobe applications use XMP sidecar files to store raw image processing data in the nested XMP namespaces of CRS and CRSS. It would be useful for exiv2 to provide read/write support for these properties by default, enabling applications which use the exiv2 library to access the data for imported raw files.

This came up as a question on the pixls.us forum,

https://discuss.pixls.us/t/xmp-sidecar-files/1423/

These properties are supported in exiftool.

http://www.sno.phy.queensu.ca/~phil/exiftool/TagNames/XMP.html#crs
