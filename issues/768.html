Reading focal length in 35mm equivalent from Canon's EXIF

Description

It seems to me that Canon cameras don't write a tag called Exif.Photo.FocalLengthIn35mmFilm, from which exiv2 tries to read focal length in 35mm equivalent.
However, exiftool can read this value from Canon's EXIF too:
Scale Factor To 35 mm Equivalent: 1.6
Focal Length : 18.0 mm (35 mm equivalent: 28.3 mm)
Lens : 18.0 - 135.0 mm (35 mm equivalent: 28.3 - 212.3 mm)

It would be nice if exiv2 was able to determine this values too.
