There is a discussion about this here: http://dev.exiv2.org/boards/3/topics/2944?r=2951#message-2951

I think it will be quite some time before this matter will get attention. Exiv2 supports Exif, IPTC, XMP and ICC metadata in a wide range of image formats. We aim to be a cross-platform, cross-format, image-standards library. As nobody has requested support for "PNG Only" metadata, there does not appear to a strong demand for this feature. For sure, to justify the "business case" to invest effort, I'd want strong evidence that this type of metadata is in common use by major applications such as the Adobe products.
