Windows writes these tags as XMP metadata - not Exif. The exifprint.cpp sample program only prints Exif metadata, not XMP.
Below is the output that I see with the current trunk version of exiv2, I don't get any error, nor a crash.
Closing this issue, feel free to re-open it with more info about an actual error/crash.
