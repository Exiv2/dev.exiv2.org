Description

Apparently, Leica does not use the LensName tag but instead:

exiftool L9997545.DNG|grep -i lens

Lens Type : Summicron-M 35mm f/2 (IV)
Lens ID : Summicron-M 35mm f/2 (IV)

Further, the actual aperture is also empty, however, this does not come
as a surprise, as on the Leica M9 this is only an estimated value,
because of which Leica chose not write this to EXIF but rather store it in
the maker notes.
