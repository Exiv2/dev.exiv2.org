Report lens model names for compact cameras

Description

In case of my compact camera Nikon Coolpix P60, no lens name is returned. The same was reported by a PowerShot G11 user today in Darktable's IRC channel.

I suggest that for fixed lenses, exiv2 returns the camera maker and model as the lens name, in case the makes doesn't provide a lens model name. This behaviour should be synchronized with exiftool and LensFun. As for LensFun, I would take care of that. This way, the exiv2/LensFun using programs would not need to implement default fallbacks of their own. Instead, the synchronisation between LensFun lookup and exiv2 information would work out-of-the-box.
